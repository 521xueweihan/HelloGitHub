# Security Policy / 安全策略

## Malicious Project Reporting / 恶意项目反馈

HelloGitHub shares entry-level open source repositories. If you identify a project listed in HelloGitHub that has been compromised, contains malware, or poses a security risk to developers, please report it immediately.
- Please do **not** open a public issue.
- Report the concern privately via GitHub's private vulnerability reporting feature or by emailing the project maintainer directly.

如果您发现 HelloGitHub 推荐的项目中包含恶意代码、存在重大漏洞或被篡改，请勿通过公开 Issue 报告，请通过私信或电子邮件直接联系维护者。

## Reporting Vulnerabilities / 漏洞汇报

If you find a security bug in our automation or web crawler scripts, please report it privately. We will coordinate a swift fix.

如果您在我们的自动化或爬虫脚本中发现安全漏洞，请通过私有渠道反馈。
